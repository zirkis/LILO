Install requirements
=====================
	
- Update the depot

```bash
sudo apt-get update
```

- Install vagrant


```bash
sudo apt-get install virtualbox

sudo apt-get install vagrant

vagrant plugin install vagrant-vbguest

```

- Install ansible

```bash
sudo apt-get install ansible #(v2.2 or more)
```
