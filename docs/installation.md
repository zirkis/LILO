Install requirement 
=====================
	
- Update the depot

		sudo apt-get update

- Install vagrant

		sudo apt-get install virtualbox

		sudo apt-get install vagrant

		vagrant plugin install vagrant-vbguest

- Install ansible

		sudo apt-get install ansible
